## Descripción de la actividad

En esta actividad se trabajará el diseño, ejecución y documentación de un sistema de copias de seguridad utilizando el software **Cobian Backup** en un entorno **Windows 11**.

El alumno deberá configurar distintos tipos de copias de seguridad, comprobar su correcta ejecución mediante los registros del sistema y realizar procesos de restauración parcial y completa de la información.

---

## Objetivos

- Comprender el funcionamiento de las copias de seguridad.
- Configurar copias completas, incrementales y diferenciales.
- Programar tareas automáticas de backup.
- Utilizar compresión y cifrado de datos.
- Restaurar información a partir de las copias de seguridad.
- Documentar correctamente todo el proceso técnico.

---

## Entorno de trabajo

- Sistema operativo: **Windows 11**
- Herramienta de copia de seguridad: **Cobian Backup**
- Almacenamiento de destino: **Disco configurado en RAID 1**

---

## Ejercicios a realizar

1. Copia de seguridad completa de una carpeta de datos.
2. Copias de seguridad incrementales programadas.
3. Restauración parcial de archivos.
4. Restauración completa de un directorio.
5. Copia diferencial comprimida y protegida con contraseña.
6. Copia comprimida y cifrada de una carpeta específica.

---

## Entrega

La entrega de la actividad deberá incluir:

- Capturas de pantalla de todo el proceso.
- Capturas de los archivos de log.
- Conclusión final.
